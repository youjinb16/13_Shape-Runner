# PaceEstimator 모듈

## 1. 기능 요약

PaceEstimator 모듈은 사용자가 입력한 목표 페이스(Pace)를 기반으로 선택한 러닝 경로의 예상 완주 시간을 계산하는 기능이다.

사용자는 자신의 목표 속도(분/km)를 입력할 수 있으며, 시스템은 경로 길이와 목표 페이스를 이용하여 예상 운동 시간을 계산한다.

이를 통해 러닝 시작 전 예상 소요 시간을 미리 확인할 수 있으며, 운동 계획 수립에 도움을 제공한다.

---

## 2. 사용 자료구조 / 알고리즘

### 사용 자료구조

#### Number

거리(distance), 페이스(pace), 예상 운동 시간(totalMinutes)을 저장하기 위해 사용하였다.

주요 연산

- 곱셈 연산
- 나눗셈 연산
- 반올림 연산

선택 이유

예상 운동 시간 계산은 단순 수치 연산으로 수행되므로 별도의 복잡한 자료구조가 필요하지 않다.

---

#### React State(useState)

사용자가 입력한 목표 페이스 값을 동적으로 관리하기 위해 사용하였다.

주요 연산

- 상태 저장
- 상태 변경
- UI 자동 갱신

선택 이유

사용자가 입력 값을 변경할 때마다 즉시 결과가 갱신되어야 하므로 React State를 활용하였다.

---

### 사용 알고리즘

#### Linear Estimation Algorithm

설명

입력된 거리와 목표 페이스를 곱하여 예상 운동 시간을 계산하는 알고리즘이다.

공식

예상 운동 시간 = 거리 × 목표 페이스

예시

- 거리 : 5km
- 목표 페이스 : 6분/km

결과

- 5 × 6 = 30분

선택 이유

러닝에서 일반적으로 사용되는 계산 방식이며 계산량이 매우 적고 직관적이다.

---
